HTML & CSS

Html & css ile istediğiniz üzerine dinamik değiştirebilir ve web siteleri yapılabilir

Web tasarımı, günümüzün en çok rağbet gören çalışma alanlarından biri olarak sürekli büyüyor. Teknolojinin hem profesyonel hem kişisel hayatlarla temasının zirveye taşındığı günümüzde, iki milyara yakın web sitesi tasarlandı. Bu sayı sürekli artarken, web tasarımcı olmak isteyenlerin sayısı da paralel olarak artmaya devam ediyor.

Dünyada kurulan ilk web sitesi, 1990 yılında CERN’de World Wide Web’in mucidi Tim Berners-Lee tarafından açıldı. İlk kez o zaman kullanılan HTML, hala web tasarımının temeli olarak önemli bir görev üstleniyor. Son çıkan HTML5 sürümüyle, oldukça komplike web sitelerinin tasarımında kullanılıyor.

CERN’de tasarlanan ilk web sitesinde de göreceğiniz gibi, başlangıçta web siteleri yalnızca metinlerden oluşuyordu ve görsel tasarım anlamında HTML’i manipüle etmek mümkün değildi. CSS ile birlikte web dünyasında bir dönüşüm yaşandı. 1996 yılında ortaya çıkan CSS ile web siteleri görsel olarak zenginleşti.

HTML Nedir?

Yazılımın diğer alanlarında olduğu gibi web tasarımda da belli başlı kabiliyetleri kazanmanın yolu işin temelini sağlam atmaktan geçiyor. 30 yıla yakın bir zamandır kullanılan HTML, web tasarımcı olmak isteyenlerin ilk öğrenmesi gereken işaretleme dilidir.

HyperText Markup Language (Zengin Metin İşaretleme Dili) ya da kısaca HTML, bir web sitesine metin, tablo, form vb. içeriklerin eklenmesini ve bunların anlamlı bir şekilde birbirine bağlanarak çalışmasını sağlıyor. Bir web sitesi tasarımı, HTML kodlarından oluşuyor.

Web sitesi oluşturmak için kullanılan bir betik dilidir. Uzantısı .html veya .htm şeklinde tanımlanır

HTML dili tag (etiket) yapısından oluşur. Genel olarak şeklinde tanımlanmış alanlar arasında işlemler yapılır. Burada dikkat edilmesi gereken nokta, etiketlerin açılması () ve kapatılması olayıdır.

HTML bir web sitenin iskeleti gibidir. İskeleti olmayan bir web site düşünülemeyeceği için html’yi kullanırız.

CSS Nedir?

CSS, “Cascading Style Sheets” kelimelerinin kısaltılmasından oluştur ve türkçe anlamı Basamaklı Stil Şablonları ya da Basamaklı Biçim Sayfaları şeklindedir

CSS en temel haliyle HTML etiketlerimizin görsel açından biçimlendirilmesini (Renk, yazı şekli, arka plan rengi, genişlik, yükseklik, pozisyon durumu vs) sağlar.

CSS’in sahip olduğu seçiciler vardır. Bunlar id ve class olarak ikiye ayrılır. HTML etiketlerinde tanımladığımız seçici isimleriyle birlikte CSS kullanarak etiketleri biçimlendiririz.

Secici ile birlikte etiketimizi seçtik. Ve font-size, color, width, height değerleri tanımlayarak html etiketimize müdahalede bulunduk.

CSS’i HTML ile oluşturduğumuz iskelet yapılarını görsel açıdan daha iyi hale getirmek için kullanırız.
